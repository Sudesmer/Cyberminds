═══════════════════════════════════════════════════════════════
  ADIM ADIM GODADDY DNS GÜNCELLEME
═══════════════════════════════════════════════════════════════

1. GoDaddy'ye giriş yapın
   → https://sso.godaddy.com

2. "My Products" → "Domains" seçin

3. "cyberminds.online" yanındaki "DNS" butonuna tıklayın

4. "Records" bölümünde:
   - Mevcut A kayıtlarını bulun
   - Her birinin yanındaki "Edit" (kalem) ikonuna tıklayın
   - "Delete" seçin
   
5. Mevcut CNAME kayıtlarını da silin (www için olanları)

6. "Add" butonuna tıklayın:
   - Type: A seçin
   - Name: @ yazın
   - Value: Yukarıdaki IP adresini girin
   - TTL: 1 Hour seçin
   - "Save" tıklayın

7. CNAME için tekrarlayın:
   - Type: CNAME seçin
   - Name: www yazın
   - Value: Site adresinizi girin
   - "Save" tıklayın

8. "Save All Changes" butonuna tıklayın

9. DNS değişikliklerinin yayılmasını bekleyin (1-48 saat)

═══════════════════════════════════════════════════════════════
  SORUN GİDERME
═══════════════════════════════════════════════════════════════

❌ "DNS_PROBE_FINISHED_NXDOMAIN" hatası:
   → DNS henüz yayılmadı, bekleyin
   → DNS kayıtlarını kontrol edin

❌ "SSL Certificate Error":
   → DNS tam yayılmadı
   → Hosting platformunda SSL'i yeniden tetikleyin

❌ Site açılmıyor:
   → Browser cache'i temizleyin
   → Incognito modda deneyin
   → VPN kapatın
