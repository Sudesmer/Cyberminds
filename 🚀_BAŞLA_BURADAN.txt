═══════════════════════════════════════════════════════════════
  🚀 3 ADIMDA DOMAIN BAĞLAMA
═══════════════════════════════════════════════════════════════

┌──────────────────────────────────────────────────────────────┐
│ ADIM 1: NETLIFY'A DEPLOY ET (5 dakika)                       │
└──────────────────────────────────────────────────────────────┘

1. Tarayıcınızda aç: https://app.netlify.com/drop

2. "cyberminds-deploy.zip" dosyasını sürükle-bırak

3. Deploy tamamlanana kadar bekle (1-2 dakika)

4. Site adını not al
   Örnek: amazing-curie-123456.netlify.app


┌──────────────────────────────────────────────────────────────┐
│ ADIM 2: CUSTOM DOMAIN EKLE (2 dakika)                        │
└──────────────────────────────────────────────────────────────┘

1. Netlify'da "Site settings" tıkla

2. "Domain management" → "Add custom domain" tıkla

3. "cyberminds.online" yaz ve "Verify" tıkla

4. "Add domain" butonuna tıkla

5. DNS ayarlarını not al (Netlify gösterecek)


┌──────────────────────────────────────────────────────────────┐
│ ADIM 3: GODADDY DNS AYARLA (5 dakika)                        │
└──────────────────────────────────────────────────────────────┘

1. GoDaddy'ye giriş yap:
   https://dcc.godaddy.com/domains

2. "cyberminds.online" → "DNS" butonuna tıkla

3. MEVCUT TÜM A VE CNAME KAYITLARINI SİL

4. YENİ KAYITLARI EKLE:

   ┌─────────────────────────────────────────────────┐
   │ A KAYDI                                          │
   ├─────────────────────────────────────────────────┤
   │ Type:  A                                         │
   │ Name:  @                                         │
   │ Value: 75.2.60.5                                │
   │ TTL:   1 Hour                                    │
   └─────────────────────────────────────────────────┘

   ┌─────────────────────────────────────────────────┐
   │ CNAME KAYDI                                      │
   ├─────────────────────────────────────────────────┤
   │ Type:  CNAME                                     │
   │ Name:  www                                       │
   │ Value: [NETLIFY-SITE-ADI].netlify.app          │
   │ TTL:   1 Hour                                    │
   └─────────────────────────────────────────────────┘

   NOT: [NETLIFY-SITE-ADI] yerine Adım 1'de not
        aldığınız site adını yazın

5. "Save" butonuna tıkla

6. 24 saat bekle (DNS yayılımı için)
